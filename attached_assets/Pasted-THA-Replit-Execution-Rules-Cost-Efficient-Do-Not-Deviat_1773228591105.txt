Objective

Simplify the basket category structure by consolidating internal ingredient categories into a smaller set of shopping categories.

This improves basket readability and reduces the number of category headers.

No logic changes to basket matching, pricing, or totals.

This is a classification and rendering change only.

--------------------------------------------------

New basket categories

Visible basket sections should be limited to:

Produce
Dairy
Eggs
Meat
Fish
Bakery
Pantry
Other

--------------------------------------------------

Category mapping rules

Existing ingredient categories must be mapped to the new basket categories.

Mapping:

fruit → produce
vegetables → produce
produce → produce
herbs (fresh) → produce

grains → pantry
oils → pantry
condiments → pantry
nuts → pantry
legumes → pantry
tinned → pantry
herbs (dried) → pantry
spices → pantry

dairy → dairy
eggs → eggs
meat → meat
fish → fish
bakery → bakery

any uncategorised → other

--------------------------------------------------

Special rule: herbs

Herbs must be split by freshness.

Fresh herbs (produce):
• coriander
• basil
• parsley
• mint
• dill

Dried herbs and spices (pantry):
• dried oregano
• cumin
• paprika
• chilli flakes
• dried coriander

If freshness cannot be determined, default to pantry.
