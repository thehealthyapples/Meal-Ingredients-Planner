T017: Subtle parallax polish (optional but recommended, should not feel “animated”)
Goal

On scroll, the meadow layers shift by only a few pixels, creating a premium feel.

Implementation (lightweight)

In OrchardBackdrop, add a useEffect that listens to scroll on the nearest scroll container.

For simplicity, listen to window scroll AND also support the main scroll container (main) if your app scrolls inside <main>.

Use requestAnimationFrame to avoid jank.

Compute:

nearY = clamp(scrollTop * 0.02, 0, var(--orchard-parallax-strength))

farY = clamp(scrollTop * 0.01, 0, var(--orchard-parallax-strength) / 2)

Set CSS variables on the backdrop root element:

--orchard-near-y: ${nearY}px

--orchard-far-y: ${farY}px

If this adds complexity, it’s acceptable to:

implement parallax only for window scrolling

or to skip parallax entirely by setting --orchard-parallax-strength: 0px

Acceptance

Effect is extremely subtle

No performance regression

Works on desktop + mobile (or gracefully no-ops)
