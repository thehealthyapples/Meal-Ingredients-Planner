Objective
Refine the THA basket into a compact category-based table with inheritance. Each category renders once with a sticky single-line header showing:

Category   Default Supermarket   Default Tier

Rows inherit those defaults unless overridden at ingredient level. If any row tier override exists in a category, the category header displays “Mixed”.

This is a layout/interaction refinement only, not a rewrite.

Scope restriction
Do NOT implement pantry-specific logic yet. Pantry should only display using the same category layout for visual consistency.

Core Behaviour

1. Category header format example:
   Produce   Tesco   Budget
   Dairy     Waitrose   Bovaer Free

2. Header fields
   • category name
   • default supermarket
   • default tier (or Mixed)

3. Rows inherit category supermarket and tier unless overridden.

4. Each row includes a Tier dropdown column.

5. Header tier behaviour
   • if all rows match category default → show tier
   • if any row tier differs → show Mixed

6. Header must be compact, single-line and sticky during scroll.

Ingredient | Variant | Tier | Qty | Match | Price | Confidence | THA Rating
