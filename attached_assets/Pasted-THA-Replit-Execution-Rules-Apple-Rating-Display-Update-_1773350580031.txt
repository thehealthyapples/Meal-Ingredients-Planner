Half scores

Support half ratings using a clipped half apple.

Examples:

2.5

Implementation rules:

• fullApples = Math.floor(score)
• if score % 1 >= 0.5 → render one half apple

The half apple must be created by clipping the existing transparent apple icon so only the LEFT half is visible.

Do NOT introduce a separate half-apple asset unless absolutely required.

Preferred approach:

• wrap the icon in a container with half width
• apply overflow hidden
• render the same apple icon inside
• clip to show only the left half

This ensures the half apple visually matches the full apples.
