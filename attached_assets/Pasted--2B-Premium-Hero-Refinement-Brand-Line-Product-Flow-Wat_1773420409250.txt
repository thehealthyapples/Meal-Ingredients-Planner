#2B - Premium Hero Refinement — Brand Line, Product Flow, Watermark, and Rating Polish
What & Why
Refine the completed premium auth/landing hero so it better reflects the full THA product story. The current version is visually strong, but it is still too narrowly framed as a one-way “kitchen → basket” flow. The hero should communicate the broader value of the product: planning, ingredient analysis, smarter baskets, and kitchen/pantry awareness. It also needs stronger brand presence via the slogan and a more visible apple watermark.

Done looks like
The left panel hero now includes the THA brand line:
- "Confidently better Choices, Simply"

The headline is updated so it reflects the fuller product story, not just cupboard-to-basket
Recommended headline:
- "Plan meals. Analyse ingredients. Build smarter baskets."

Supporting copy is updated accordingly so the story is:
- planning
- ingredient intelligence
- grocery decision support
- lower effort

The lower hero visual is no longer framed as a single one-way cupboard → basket flow
Instead, it should feel like a connected product flow or ecosystem, such as:
- Pantry / Kitchen
- Plan
- Analyse
- Basket

The current single directional arrow/connector is replaced with a broader connected-flow treatment that better reflects the app

The apple watermark behind the hero text is made more visible and intentional
- increase opacity moderately
- increase size if needed
- reposition so it is clearly legible behind the text area
- remain decorative only

The apple-rating display on the basket rows is polished so it clearly reads as THA apple ratings, not star-like marks
- use the existing tha-apple.png asset
- improve sizing, spacing, alignment, and clarity
- recreate the rating presentation if needed

The right-hand login/register form remains completely untouched
No backend routes, schemas, or unrelated pages are changed
