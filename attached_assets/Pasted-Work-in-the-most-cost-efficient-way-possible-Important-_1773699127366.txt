Current gap:

Required behaviour:

- roast potatoes
- cauliflower cheese
- gravy
- Yorkshire puddings
- braised cabbage

THA should allow that part to be resolved from one of these sources:
1. My Meals existing recipes
2. internal THA recipe search / local recipe data
3. web recipe/source search
4. simple basic item / wholefood item
5. fresh / frozen / basic option where appropriate for simple veg or sides

UI requirements:
- For each entered meal part, show a lightweight source selection flow.
- At minimum provide actions such as:
  - Use My Meals match
  - Search THA
  - Search web
  - Use basic item
- For simple wholefood items, allow practical options such as:
  - Fresh
  - Frozen
  - Basic wholefood entry
