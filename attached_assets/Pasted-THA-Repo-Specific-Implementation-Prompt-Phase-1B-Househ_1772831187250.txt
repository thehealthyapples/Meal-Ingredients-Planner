Write pattern example

For any shared create flow, the intended pattern is:

const householdId = await getHouseholdForUser(userId);

await db.insert(shoppingList).values({
  userId,
  householdId,
  ...rest
});

For any shared read flow:

return db.query.shoppingList.findMany({
  where: eq(shoppingList.householdId, householdId)
});

For update/delete by id, scope by household ownership.
