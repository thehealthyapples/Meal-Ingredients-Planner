Objective

Fix the UI regressions introduced in the “Calm Orchard” redesign:

Remove all pastel colour blocks (pink/green/blue card fills, tinted headers, category colour panels) that no longer match the brand and look amateur.

Ensure all cards/widgets use the Calm Orchard palette: warm cream + sage + subtle sun accent only.

Keep the orchard/meadow backdrop behind the page, but never inside cards.

Improve “My Meals” page widget/filter area so it feels calm and premium (reduce clutter visually) without removing any functionality.

Hard constraints

UI only: do NOT change API calls, business logic, matching logic, totals, exports, routing behavior, or data structures.

Do NOT remove filter functionality—only restyle and reorganize presentation.

Keep readability high: cards remain solid bg-card, tables remain clean.

T018: Remove pastel backgrounds from Meal Cards (and any cards site-wide)
Problem

Meal cards currently have large pastel blocks (pink/green) behind icons/titles that clash with the new palette.

Required outcome

All cards use:

background: bg-card

border: border border-border

no pastel fills / gradients inside the card body/header

Steps

Locate the Meal Card component(s). Likely files include (search for “MealCard”, “meal card”, or where cards are rendered on My Meals):

client/src/pages/meals-page.tsx

client/src/components/meals/*

client/src/components/cards/*

Search for and remove/replace any of these inside card markup:

bg-pink-*, bg-green-*, bg-blue-*, bg-purple-*, bg-yellow-* (except brand sun accent usage)

from-* to-* gradients

opacity-* overlays used to tint the header area

inline styles setting background colours based on category/type

Replace with a clean card surface:

Card wrapper: bg-card border border-border shadow-none rounded-[var(--radius)]

Card header area: no fill; optionally add a subtle divider border-b border-border

If there is an “icon tile” behind the icon, replace it with:

bg-accent (very light sage wash)

text-primary

small square/pill (not a large half-card block)

Acceptance

No large pastel blocks remain in meal cards.

Cards look consistent across categories.

Only Calm Orchard palette is visible.
